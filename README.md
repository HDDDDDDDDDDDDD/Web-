# Web登录应用

这是一个使用Python Flask框架开发的简单Web登录应用。

## 功能特性

- 用户登录界面，包含用户名和密码输入
- 点击登录后跳转到结果显示页面
- 美化的用户界面，支持背景图片
- 响应式设计，适配不同屏幕尺寸

## 文件结构

```
lingma/
├── app.py              # 应用主程序
├── requirements.txt    # 依赖包列表
├── README.md           # 说明文档
├── static/             # 静态资源目录
│   └── *.webp          # 背景图片
└── templates/          # HTML模板目录
    ├── login.html      # 登录页面模板
    └── result.html     # 结果显示页面模板
```

## 运行环境

- Python 3.6+
- Flask 2.0.1

## 安装和运行

1. 安装依赖：
   ```
   pip install -r requirements.txt
   ```

2. 运行应用：
   ```
   python app.py
   ```

3. 在浏览器中访问 `http://127.0.0.1:5000`

## 使用说明

1. 在登录页面输入用户名和密码
2. 点击登录按钮
3. 页面将跳转到结果显示页面，显示输入的用户名和密码
4. 可以点击"返回登录页面"按钮回到登录页面

## 注意事项

- 这是一个演示应用，不包含实际的用户认证逻辑
- 密码以明文形式显示，仅用于演示目的
